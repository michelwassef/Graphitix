  • ✓ UI Integration: Axis control panel supports adding, editing, and removing break segments

How to Use:

  1. Open the Line & Area Charts workspace
  2. Load example data or import your own dataset
  3. Click on the X or Y axis to open the axis controls panel
  4. Click the "Break axis" button
  5. Enable "Broken Axis" checkbox
  6. Click "+ Add Break" to add break segments
  7. Enter start and end values for each break (e.g., 10 → 90 to hide values between 10 and 90)
  8. The axis will automatically redraw with gaps where breaks are defined

Technical Details:

Example Break Configuration: { x: { enabled: true, segments: [ { start: 0, end: 5 }, { start: 95, end: 100 } ] }, y: { enabled: true, segments: [ { start: 0, end: 20 }, { start: 80, end: 200 } ] } } This configuration will: - Show X values 0-5 and 95-100 with a gap between - Show Y values 0-20 and 80-200 with a gap between - Skip drawing ticks/gridlines in the gap ranges - Proportionally allocate pixel space to each segment
